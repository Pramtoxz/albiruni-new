# Logo Daycare

## Cara Mengganti Logo

Untuk mengganti logo di PDF Menu Mingguan:

1. Siapkan file logo Anda dalam format PNG atau SVG
2. Rename file menjadi `daycare-logo.png` atau `daycare-logo.svg`
3. Letakkan file di folder `public/images/`
4. Logo akan otomatis muncul di PDF

## Spesifikasi Logo

- Format: PNG atau SVG
- Ukuran: 70x70 pixels (atau rasio 1:1)
- Nama file: `daycare-logo.png` atau `daycare-logo.svg`

## Catatan

Jika tidak ada file logo, sistem akan menggunakan logo default SVG yang sudah disediakan.
